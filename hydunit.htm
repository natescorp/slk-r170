Гидравлический блок


Расположенный в багажнике за правой боковой панелью,
гидравлический блок создает давление для распределения по рабочим цилиндрам.
Два электромагнитных клапана, расположенных на держателе для гидравлического блока,
распределяют жидкость по гидравлическим цилиндрам в зависимости от приведения в действие крыши.
