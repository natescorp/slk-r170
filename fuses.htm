Расположение предохранителей

Главный блок предохранителей
в моторном отсеке — содержит
предохранители для
внутреннего оборудования. 		Дополнительный блок
предохранителей в багажнике
— содержит предохранители
для системы запирания
и управления крышей. 		Дополнительный блок
предохранителей за выключателем
света — содержит предохранители
для освещения.
