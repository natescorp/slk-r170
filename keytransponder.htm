Drive Authorization System (DAS)

The R170 is equipped with a new
Drive Authorization System (DAS)
for increased operating reliability.
The system is no longer armed
and disarmed via the infra-red
remote control as with previous
models, but by data transfer
from the ignition lock, The key
contains a transponder with an
automatic code modulator. This
is supplied with energy by a coil
on the ignition lock when the
ignition is switched on, and in turn,
emits a signal which is recieved by
the coil and transmitted to the central
locking unit. If the correct code is
recieved, a signal is sent to the engine
management system to allow starting.

The advantage of this system is that
a signal can be sent to the immobilizer
regardless of the condition of the battery
in the key, and independently of the
remote control system.
